Title: Study Buddy
Ownership: Abigail Liskey
Course/Project: Database Design & Application
University/Institution: James Madison University
Written in: HTML, CSS, PostgreSQL, PgAdmin, Javascript, SQL
Created: 2013/05

Description: Tool that enables students (parents) to compare individual student performance

to peers within the local division or across the state.  Offers activites/lesson
plans based
on learning needs for remediation.
  This web-application was an award winning app recognized
by the Virginia Department of Education and the Virginia Longitudinal Database System as well
as Center for Innovative Technology.

Commited to GitHub 2014/11/5
